Welcome to ZigZag!

ZigZag is a game built in C using an ARM7 processor to create a fully responsive game!
While coded in C, the game is built to run with an FPGA device and renders a screen to play on.

Features include:
- Game over!
- High Score
- Responsive controls
- Gems
- And more!
